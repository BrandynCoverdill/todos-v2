# React Template

This template provides a fast way to start creating new apps with React.

## Installed Packages

This template has the following packages installed:

- @emotion/react
- @emotion/styled
- @mui/icons-material
- @mui/lab
- @mui/material
- react-router-dom

## Use / Run template

To use/run this template, run the following commands:

```
npm i
npm run dev
```
